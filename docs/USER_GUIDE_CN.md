## 应用概览

PKU Course Desktop 是一款跨平台桌面应用，用于从 PKU 教学网（`course.pku.edu.cn`）下载课程录播视频。主要功能包括：

- 内置浏览器，自动检测页面中的录播视频
- 一键下载，实时显示下载进度（百分比、速度、剩余时间）
- 批量下载队列，支持并发下载控制
- 自动将 m3u8 流转码为 MP4
- 可选音频提取功能（支持 MP3 / AAC / WAV 格式）
- 支持浅色 / 深色主题切换

支持平台：Windows 10+、macOS 11+、Ubuntu 24+。

## 环境准备

在使用本应用前，你需要先在系统中安装 **FFmpeg**。FFmpeg 用于以下功能：

- 将 m3u8 视频流转码为 MP4 文件
- 从视频文件中提取音频

### macOS

```bash
brew install ffmpeg
```

### Ubuntu / Debian

```bash
sudo apt update
sudo apt install ffmpeg
```

> **Linux 用户提示：** 如果你想在内嵌浏览器中直接播放 HLS（m3u8）视频，建议同时安装 GStreamer 插件：
> ```bash
> sudo apt install gstreamer1.0-plugins-base gstreamer1.0-plugins-good gstreamer1.0-plugins-bad gstreamer1.0-libav
> ```

### Windows

1. 从 [gyan.dev](https://www.gyan.dev/ffmpeg/builds/) 下载 FFmpeg（推荐下载 `ffmpeg-git-essentials.7z`）。
2. 将压缩包解压到 `C:\ffmpeg` 目录。
3. 将 `C:\ffmpeg\bin` 添加到系统的 `PATH` 环境变量中。

安装完成后，打开命令提示符或 PowerShell，运行以下命令验证：

```bash
ffmpeg -version
```

## 安装应用

1. 从 [Releases](https://github.com/morganchenjp/PKU-Course-Desktop/releases) 页面下载适合你操作系统的安装包。
2. 按照常规方式安装：
   - **Windows**：运行 `.msi` 安装程序。
   - **macOS**：打开 `.dmg` 文件，将应用拖入 `Applications` 文件夹。
   - **Linux**：将 AppImage 文件设为可执行（`chmod +x PKU-Course-Desktop.AppImage`）后直接运行。
3. 启动应用。

## 首次启动与登录

首次打开 PKU Course Desktop 时：

1. 应用会自动在内置浏览器中打开北大教学网（`course.pku.edu.cn`）。
2. 点击 **"校园卡用户"**，使用你的 IAAA 统一身份认证账号登录。
3. 登录成功后会进入教学网主页。

> **建议：** 为了获得最佳的浏览体验，建议将应用窗口最大化使用。

## 查找并下载视频

### 第一步 — 进入课程页面

像平常使用教学网一样浏览：

1. 从课程列表中点击进入一门课程。
2. 点击 **"课堂实录"**。
3. 点击你想下载的那次课堂实录对应的日期。

### 第二步 — 视频自动检测

当视频播放页面加载完成后，应用会自动检测到视频信息。你会在视频播放器下方看到三个按钮：

- **下载视频**
- **复制链接**
- **刷新**

### 第三步 — 开始下载

点击 **"下载视频"** 按钮，应用会执行以下操作：

1. 创建一个下载任务。
2. 如果视频是 m3u8 流格式，应用会先下载流数据，然后自动转码为 MP4。
3. 在下载管理面板中显示实时下载进度。

> **提示：** 如果你在设置中开启了 **"自动开始下载"**，视频被检测到后会自动加入队列并开始下载，无需手动点击。

## 下载管理

点击顶部导航栏的 **"下载管理"** 按钮，可以打开下载管理面板。

### 任务状态说明

| 状态 | 含义 |
|------|------|
| **等待中** | 已加入队列，正在等待空闲下载槽位 |
| **下载中** | 正在下载，显示实时进度 |
| **已完成** | 下载成功完成 |
| **失败** | 下载失败（可在任务卡片中查看具体错误信息） |

### 操作说明

- **筛选任务**：使用顶部的标签页（全部 / 下载中 / 等待中 / 已完成 / 失败）按状态筛选任务。
- **清空已完成**：点击 **"清空已完成"** 按钮，移除所有已完成的任务记录。
- **清空全部**：点击 **"清空全部" 按钮**，清空所有任务记录。此操作仅删除任务列表，不会删除已下载的文件。

### 进度信息

任务下载过程中，每张任务卡片会显示：

- **进度条** 和精确百分比
- **下载速度**（例如：`2.5 MB/s`）
- **预计剩余时间**（ETA）

## 设置说明

点击顶部导航栏的 **"设置"** 按钮，打开设置面板。

### 下载设置

| 设置项 | 说明 |
|--------|------|
| **下载路径** | 视频文件保存的文件夹。留空则默认保存到系统的"下载"文件夹。 |
| **文件命名规则** | 下载文件的命名方式。可用变量：`{courseName}`（课程名）、`{subTitle}`（副标题）、`{lecturerName}`（讲师名）、`{date}`（日期）、`{index}`（序号）。 |
| **最大并发下载数** | 同时下载的视频数量上限（1–2）,缺省为1, 教学网服务器端有限制。 |
| **自动开始下载** | 开启后，检测到视频时自动添加到队列并开始下载。 |

### 视频设置

| 设置项 | 说明 |
|--------|------|
| **默认视频质量** | 优先选择的视频流质量（最高质量 / 高质量 / 中等质量 / 低质量）, 由于教学网限制，这个选项目前无实际价值。 |
| **同时提取音频文件** | 开启后，视频下载完成后自动从中提取音频。 |
| **音频格式** | 选择 MP3、AAC 或 WAV 格式。仅在开启"同时提取音频文件"后显示。|

### 保存设置

修改设置后，必须点击底部的 **"保存设置"** 按钮才会生效。如果想放弃未保存的修改，可以点击 **"恢复默认"**。

## 音频提取

音频提取功能适合以下场景：

- 在不支持视频播放的设备上听课程录音。
- 将音频上传到 AI 转写工具（如阿里["通义听悟"](https://tingwu.aliyun.com/home)，自动生成课堂笔记。

### 如何开启

1. 打开 **设置** 面板。
2. 开启 **"同时提取音频文件"** 开关。
3. 选择 **音频格式**：
   - **MP3** — 兼容性最好，中等音质 VBR（约 185 kbps）
   - **AAC** — 相同文件大小下音质更好（128 kbps）
   - **WAV** — 无损未压缩，文件体积最大
4. 点击 **"保存设置"**。

### 输出结果

视频下载完成后，应用会在后台调用 FFmpeg，在同一文件夹下生成一个同名音频文件。例如：

```
Download/
├── 高等数学 - 第3周 - 张教授.mp4
└── 高等数学 - 第3周 - 张教授.mp4.mp3   <-- 提取的音频
```

> **Windows 用户提示：** FFmpeg 在后台静默运行，不会弹出任何命令行窗口。

## 常见问题排查

### 提示 "FFmpeg not found" 错误

说明系统中未安装 FFmpeg，或者 FFmpeg 不在系统 `PATH` 中。请按照 [环境准备](#环境准备) 部分的说明安装 FFmpeg，然后重启应用。

### 下载任务立即显示 "失败"

常见原因：

- **会话过期**：你可能已登出。重新进入课程页面刷新会话即可。
- **网络问题**：请检查网络连接是否正常。
- **服务器错误（HTTP 500）**：服务器拒绝了请求。尝试刷新视频页面后重新点击下载。

### 视频播放器黑屏或无法加载

- 确认你已正常登录（IAAA 会话可能已过期）。
- 尝试点击视频下方的 **"刷新"** 按钮。
- Linux 用户请确认已安装 GStreamer 插件（见 [环境准备](#环境准备)）。

### 下载进度一直卡在 0%

- 检查是否处于代理或防火墙环境，导致无法访问下载地址。
- 尝试在设置中将 **最大并发下载数** 调整为 1。

### 点击 "下载视频" 后应用窗口变白

这是旧版本的已知 bug。如遇到此问题，请更新到最新版本。

## 常见问题 FAQ

**Q: 下载的文件保存在哪里？**

默认保存在系统"下载"文件夹中。你可以在 **设置 → 下载路径** 中修改保存位置。

**Q: 可以同时下载多个视频吗？**

可以。在设置中将 **最大并发下载数** 设为 2，额外的视频会进入队列，有空闲槽位时自动开始下载。

**Q: 支持断点续传吗？**

目前暂不支持。如果下载中断，需要重新开始下载。

**Q: 我的登录密码安全吗？**

安全。应用使用系统自带的 WebView 引擎（Linux 为 WebKitGTK，Windows 为 WebView2，macOS 为 WKWebView）。你的密码由北大 IAAA 登录页面直接处理，应用不会读取或存储你的密码。

**Q: 在校园 VPN 环境下可以使用吗？**

可以。只要你的系统能够访问 `course.pku.edu.cn`，应用就能正常工作。

**Q: 下载的 MP4 文件在某些播放器中无法播放？**

建议使用 VLC 或 MPV 等现代播放器。如果是从 m3u8 转换而来的文件，编码格式通常为 H.264/AAC，主流播放器都支持。
